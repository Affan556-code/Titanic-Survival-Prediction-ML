# Titanic Survival Prediction using Machine Learning

## Overview
This project uses Machine Learning techniques to predict whether a passenger survived the Titanic disaster.

The project demonstrates complete ML workflow including:
- Data preprocessing
- Handling missing values
- Label Encoding
- Model training
- Prediction and evaluation

---

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithm
- Logistic Regression

---

## Dataset
Titanic dataset from Kaggle.

---

## Workflow
1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Feature Selection
5. Model Training
6. Prediction
7. Accuracy Evaluation

---

## Result
The machine learning model successfully predicted passenger survival with good accuracy.
